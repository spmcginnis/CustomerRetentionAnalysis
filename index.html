Overview

A project to demonstrate a workflow for a customer retention study. The data was created with a hypothetical coffee shop in mind to measure customer retention over a period of time. The initial data fields included customers and their visits to the coffee shop. Certain parameters (such as length of study, number of customers, number of visits per day, etc.) can be adjusted in the data generation phase in order to mimic different patterns. See the individual reports below for details.

Stack

  • Python to generate the data, with the pyodbc library to pass SQL commands to the database.
  • MS SQLExpress Database for local data storage.
  • HTML5 with Bootstrap for display.
